# Hobby-E-Commerce

Project 3: E-Commerce Site.

## Description

Hobby Shop is a E-Commerce website that sells game merchandice.

We wanted to showcase our our skils and understanding for web development practices and topics, such as:

- MongoDB
- Render
- JWT
- GraphQL
- Tailwind
